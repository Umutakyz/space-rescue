PROJECT OBJECTIVE AND SCOPE 

•Objective: To develop a 2D score-based arcade game with an infinite loop using the Unity Game Engine and C# programming language. 

Key Features: 

Player movement mechanics (Mouse/Keyboard integration). 

Dynamic enemy generation (Random Spawning). 

Collision system and physics interactions. UI (User Interface) management and game flow control with Singleton design pattern.
